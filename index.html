HTML: A Estrutura

O HTML (HyperText Markup Language) é a linguagem de marcação que forma a espinha dorsal de toda a web. Ele diz ao navegador o que é cada pedaço de conteúdo (um título, um parágrafo, uma imagem, etc.) e como ele deve ser estruturado.

Exemplo Básico de Sintaxe HTML

O HTML usa "tags" (etiquetas) envoltas em < e >. A maioria das tags tem uma de abertura (ex: <p>) e uma de fechamento (ex: </p>).


<!-- Este é um comentário em HTML. O navegador o ignora. -->
<!DOCTYPE html>
<html lang="pt-BR">
  <head>
    <meta charset="UTF-8">
    <title>Meu Site</title>
  </head>
  <body>
    <h1>Este é um Título Principal</h1>
    <p>Este é um parágrafo.</p>
  </body>
</html>

Tags de Formatação de Texto (Inline)

Estas tags são usadas "em linha" (inline) para estilizar ou dar significado a partes específicas de um texto, sem quebrar a linha.

<strong> e <b> (Importante / Negrito)

<strong> é semântico: indica que o texto tem grande importância ou seriedade. Os navegadores o renderizam em negrito.

<b> é apenas estilístico: indica que o texto deve ser em negrito (bold), sem implicar importância.

<p>Use <b>apenas para estilo</b>.</p>

<em> é semântico: indica ênfase (emphasis) no texto, como uma mudança de tom. Os navegadores o renderizam em itálico.

<i> é estilístico: indica que o texto deve ser em itálico (italic), como para um termo estrangeiro ou nome de navio.

<p>Você <em>precisa</em> ver isso.</p>
<p>O termo <i>"semântica"</i> é importante.</p>

Tags de Links e Imagens

A Tag <a> (Âncora/Link)

Cria um hyperlink. O atributo href (hypertext reference) define o destino do link. O atributo target="_blank" faz o link abrir em uma nova aba.

<!-- Link para um site externo -->
<a href="https.google.com" target="_blank">Visite o Google</a>

<!-- Link para uma seção interna da página -->
<a href="#css">Ir para a Seção CSS</a>

A Tag <img> (Imagem)

Incorpora uma imagem na página. É uma tag vazia. src é a fonte (source) da imagem. alt (alternative text) é crucial para acessibilidade, descrevendo a imagem para leitores de tela.

<img src="https/via.placeholder.com/150x50" 
     alt="Um retângulo cinza de 150 por 50 pixels">

Tags de Formulário (Expandido)

Além de <input>, temos outras tags importantes para coletar dados do usuário.

A Tag <textarea>

Cria uma caixa de texto de múltiplas linhas, ideal para mensagens ou comentários.

<label for="comentario">Comentário:</label>
<textarea id="comentario" rows="4" cols="50"></textarea>

Tags <select> e <option>

Cria uma lista suspensa (dropdown). <select> é o contêiner e cada <option> é um item da lista.

<label for="classe">Escolha sua classe:</label>
<select id="classe" name="classe">
  <option value="guerreiro">Guerreiro</option>
  <option value="mago">Mago</option>
</select>

Tags <fieldset> e <legend>

Agrupam campos de formulário relacionados. <fieldset> desenha uma caixa ao redor, e <legend> dá um título (legenda) para essa caixa.

<fieldset>
  <legend>Informações Pessoais</legend>
  <label for="nome">Nome:</label>
  <input type="text" id="nome">
</fieldset>

Tags de Estrutura Semântica

Estas tags definem a estrutura e o significado das seções da sua página, substituindo o uso excessivo de <div>.

  • <header>: O cabeçalho da página ou de uma seção (contém logo, navegação).
  • <nav>: Contém os links de navegação principais.
  • <main>: Define o conteúdo principal e único da página.
  • <section>: Define uma seção temática de conteúdo.
  • <article>: Define um conteúdo independente e auto-contido (ex: um post de blog, um produto).
  • <aside>: Define conteúdo lateral (sidebar), como links relacionados ou biografias.
  • <footer>: O rodapé da página ou de uma seção (contém copyright, links de contato).

CSS: O Estilo

O CSS (Cascading Style Sheets) é a linguagem que usamos para estilizar o HTML. Se o HTML é o esqueleto, o CSS é a pele, as roupas e as cores. Ele controla fontes, cores, espaçamentos, posicionamento e até mesmo animações.

Como Adicionar CSS

Existem três maneiras de adicionar CSS a uma página, sendo a "Externa" a mais recomendada.

  • Externo (External): Um arquivo .css separado, linkado no <head>. (É o que estamos fazendo neste projeto).
  • Interno (Internal): Usando a tag <style> dentro do <head> do HTML.
  • Na Linha (Inline): Usando o atributo style="..." diretamente em uma tag HTML. (Não recomendado).
<!-- 1. Externo (Recomendado) -->
<link rel="stylesheet" href="style.css">

<!-- 2. Interno -->
<style>
  body { background-color: #121212; }
</style>

<!-- 3. Inline (Evite) -->
<p style="color: red;">Texto vermelho.</p>

Seletores Essenciais

Seletores são a forma como o CSS "encontra" os elementos HTML para estilizar.

Básicos (Tipo, Classe, ID)

  • Tipo: Seleciona todas as tags de um tipo. (ex: p)
  • Classe: Seleciona todos os elementos com class="...". (ex: .card)
  • ID: Seleciona o *único* elemento com id="...". (ex: #meu-formulario)

Combinadores

  • Descendente (espaço): .card p (Seleciona todo <p> *dentro* de .card).
  • Filho Direto (>): .card > h3 (Seleciona todo <h3> que é *filho direto* de .card).
  • Irmão Adjacente (+): h3 + p (Seleciona o <p> que vem *imediatamente após* um <h3>).

Pseudo-classes

Seleciona elementos com base em seu estado ou posição.

  • :hover: Quando o mouse está sobre o elemento.
  • :focus: Quando o elemento (ex: input) está focado.
  • :nth-child(n): Seleciona o n-ésimo filho (ex: li:nth-child(odd) para linhas ímpares).
/* O botão fica azul escuro quando o mouse passa por cima */
.button-primary:hover {
  background-color: var(--cor-secundaria);
}

/* A 3ª linha da lista fica azul */
li:nth-child(3) {
  color: var(--cor-primaria);
}

Pseudo-elementos

Estiliza uma *parte* específica de um elemento.

  • ::before: Insere conteúdo *antes* do conteúdo do elemento.
  • ::after: Insere conteúdo *depois* do conteúdo do elemento.
/* Adiciona uma seta "➡️" antes de cada item da lista */
.modern-list li::before {
  content: "➡️ ";
}

O Box Model (Modelo de Caixa)

Todo elemento HTML é uma caixa. O Box Model é como calculamos seu tamanho: Conteúdo + Padding (espaço interno) + Border (borda) + Margin (espaço externo).

.minha-caixa {
  margin: 20px;  /* Espaço fora da borda */
  padding: 30px; /* Espaço entre a borda e o conteúdo */
  border: 5px solid white;
  background-color: var(--cor-primaria);
}

Propriedade 'display'

A propriedade display é a mais importante do CSS para controlar layout. Ela define como um elemento se comporta.

  • display: block;: (Padrão para <div>) Ocupa 100% da largura, começa em nova linha.
  • display: inline;: (Padrão para <span>) Ocupa só o espaço necessário, fica na mesma linha.
  • display: inline-block;: Um híbrido. Fica na mesma linha (inline) mas aceita width e height (block).
  • display: flex;: Ativa o Flexbox (veja abaixo).
  • display: grid;: Ativa o Grid Layout (veja abaixo).
  • display: none;: Esconde o elemento completamente (o JS usa isso).

Propriedade 'position'

A propriedade position controla como um elemento é posicionado, permitindo "quebrar" o fluxo normal.

position: relative e absolute

relative: O elemento fica "relativo" à sua posição original. Usado como âncora para filhos "absolutos".

absolute: O elemento é removido do fluxo e posicionado em relação ao "pai relativo" mais próximo.

.pai-relativo {
  position: relative;
  height: 100px;
}
.filho-absoluto {
  position: absolute;
  bottom: 10px;
  right: 10px;
}
Eu sou o contêiner PAI (position: relative).
Eu sou o FILHO (position: absolute) e estou "preso" a 10px do canto inferior direito do meu pai.

position: fixed e sticky

fixed: O elemento é posicionado em relação à janela do navegador (ex: um menu que fica fixo no topo). Fica "preso" mesmo com a rolagem.

sticky: Um híbrido. Ele se comporta como static (normal) até que a rolagem o faça atingir um ponto específico (ex: top: 0), então ele se torna fixed.

/* Este menu ficaria preso no topo da tela */
.menu-fixo {
  position: fixed;
  top: 0;
  width: 100%;
}
/* Este título de seção "grudaria" no topo ao rolar */
h2 {
  position: sticky;
  top: 0;
}

Unidades de Medida

  • Absolutas: px (pixels). Tamanho fixo.
  • Relativas à Fonte: em (relativo ao pai), rem (relativo à raiz <html>). Ótimas para acessibilidade.
  • Relativas à Viewport: % (do pai), vw (da largura da janela), vh (da altura da janela).

Design Responsivo (Media Queries)

Usamos @media para aplicar estilos diferentes dependendo das características do dispositivo, como o tamanho da tela.

/* Estilos padrão (desktop primeiro) */
.container {
  width: 900px;
}

/* Se a tela for MENOR que 768px... */
@media (max-width: 768px) {
  .container {
    /* ...o container ocupa 100% da largura. */
    width: 100%;
  }
}

JavaScript: A Interatividade

O JavaScript (JS) é a linguagem de programação do navegador. Se o HTML é o esqueleto e o CSS é a pele, o JS são os músculos e o cérebro. Ele permite que a página reaja a ações do usuário (eventos), manipule o HTML/CSS (o DOM) e dê 'vida' à página.

Onde Adicionar JavaScript

  • Externo (Recomendado): Em um arquivo .js, linkado no final do <body> ou no <head> com o atributo defer.
  • Interno: Dentro de uma tag <script> no HTML.
<!-- 1. Externo com 'defer' (Recomendado) -->
<script src="script.js" defer></script>

<!-- 2. Interno (Evite para scripts grandes) -->
<script>
  console.log("Olá do script interno!");
</script>

Variáveis, Tipos de Dados e Operadores

Variáveis (let vs. const)

Variáveis são "contêineres" para armazenar dados. Use const por padrão. Se você precisar que o valor mude, use let.

Tipos de Dados Primitivos

  • String: Texto (ex: "Olá mundo")
  • Number: Números (ex: 101 ou 3.14)
  • Boolean: Verdadeiro ou falso (ex: true ou false)
  • Null: Representa a ausência intencional de um valor.
  • Undefined: Representa uma variável que não foi definida.

Operadores

Símbolos que executam operações em valores e variáveis.

  • Aritméticos: + (soma), - (subtração), * (multiplicação), / (divisão).
  • Atribuição: = (atribui um valor, ex: let x = 10;).
  • Comparação (Importante):
    • == (Igualdade "solta"): Compara apenas o valor. (ex: 5 == "5" é `true`) - **Evite!**
    • === (Igualdade "estrita"): Compara o valor E o tipo. (ex: 5 === "5" é `false`) - **Use este!**
    • != (Diferente "solto") e !== (Diferente "estrito").
    • > (maior que), < (menor que), >=, <=
let idade = 25;
let pontos = 10;
let total = pontos * 2; // total será 20

if (idade > 18) {
  console.log("Maior de idade");
}

Funções

Funções são blocos de código reutilizáveis que executam uma tarefa. Elas podem receber dados (parâmetros) e retornar um valor.

// 1. Declaração de Função
function saudacao(nome) {
  return "Olá, " + nome + "!";
}

// 2. Chamando a Função
let mensagem = saudacao("Ana"); // mensagem será "Olá, Ana!"
console.log(mensagem);

Objetos (Básico)

Objetos são coleções de pares chave/valor, usados para agrupar dados relacionados.

// Objeto Literal
const usuario = {
  nome: "Carlos",
  idade: 30,
  isAdmin: true,
  saudar: function() {
    console.log("Oi, eu sou o " + this.nome);
  }
};

// Acessando propriedades
console.log(usuario.nome); // "Carlos"
usuario.saudar(); // "Oi, eu sou o Carlos"

Condicionais (if / else)

Executa blocos de código diferentes com base em uma condição (booleana).

let hora = new Date().getHours(); // Pega a hora atual (0-23)

if (hora < 12) {
  console.log("Bom dia!");
} else if (hora < 18) {
  console.log("Boa tarde!");
} else {
  console.log("Boa noite!");
}

Clique no botão para executar um if/else com base na hora atual.

Loops (for)

Loops são usados para executar uma ação repetidas vezes. O loop for é o mais comum.

let frutas = ["Maçã", "Banana", "Laranja"];

// O loop começa com i=0; continua ENQUANTO i < 3; incrementa i a cada rodada.
for (let i = 0; i < frutas.length; i++) {
  console.log(frutas[i]); // "Maçã", "Banana", "Laranja"
}

Interagindo com o DOM

O DOM (Document Object Model) é a representação em árvore do seu HTML. O JS pode manipular o DOM para alterar dinamicamente o conteúdo, estilo ou estrutura da página.

1. Encontrando Elementos

  • document.getElementById("meu-id"): Rápido e específico, pega pelo ID.
  • document.querySelector(".minha-classe"): Moderno e flexível, pega o *primeiro* elemento que bate com o seletor CSS (pode ser #id, .classe, tag, etc).
  • document.querySelectorAll(".minha-classe"): Pega *todos* os elementos que batem com o seletor.

2. Alterando Conteúdo (innerHTML vs. textContent)

textContent insere texto puro (seguro). innerHTML insere HTML (pode ser inseguro se vier do usuário, mas é útil para renderizar tags).

const titulo = document.querySelector('h2');

// Apenas texto, seguro.
titulo.textContent = "Novo Título (Texto)";

// Interpreta o HTML, mas pode ser inseguro.
titulo.innerHTML = "<i>Novo Título (HTML)</i>";

Este texto vai mudar.

3. Alterando Estilos (style)

Você pode alterar o CSS "inline" de um elemento usando a propriedade .style. Note que propriedades CSS com hífen (ex: background-color) viram "camelCase" (ex: backgroundColor).

const elemento = document.querySelector('#demo-style-box');
elemento.style.backgroundColor = '#00aaff';
elemento.style.color = '#000';
Esta caixa vai mudar de cor.

4. Alterando Atributos (setAttribute)

Você pode mudar atributos de um elemento, como o src de uma imagem ou o href de um link.

const botao = document.querySelector('#btn-demo-disabled');
// Adiciona o atributo 'disabled', desativando o botão
botao.setAttribute('disabled', 'true');

Eventos Comuns

Eventos são ações que o JS pode "ouvir". Usamos .addEventListener(evento, funcao).

  • 'click': O usuário clica em um elemento.
  • 'submit': O usuário envia um formulário. (Lembre-se do event.preventDefault()).
  • 'mouseover': O mouse passa por cima de um elemento.
  • 'keydown': O usuário pressiona uma tecla.
  • 'DOMContentLoaded': O HTML foi completamente carregado. (Nosso script principal usa este).
